# Chatroom Template

終端機風格直播聊天室模板。

原本只是因為聊天室模板搞丟了，所以乾脆重做一份，最後順手整理後公開到 GitHub。

目前提供 Streamlabs 與 OneComme 9.0 版本，並包含直式與橫式版型。

---

## 功能特色

- 終端機風格介面
- 支援 Streamlabs
- 支援 OneComme 9.0
- 提供直式聊天室版型
- 提供橫式 Wide Overlay 版型
- 支援聊天室徽章顯示
- 支援 Twitch / YouTube 表情顯示
- 自動限制聊天室訊息數量
- 支援聊天室清除事件
- 支援訊息刪除事件
- 開放原始碼，可自行修改

---

## 內含模板

### Chatroom Sync V1.0

標準直式聊天室。

支援平台：

- Streamlabs

### Chatroom Sync wide-overlay V1.0

適合遊戲直播使用的橫式版型。

支援平台：

- Streamlabs

### onecomme Chatroom Sync V1.0

OneComme 專用版本。

支援平台：

- OneComme 9.0

---

## 使用方式

### Streamlabs

1. 下載需要使用的模板資料夾
2. 將對應檔案匯入 Streamlabs
3. 設定 Browser Source 或 Custom Widget
4. 完成

### OneComme

1. 下載 OneComme 版本模板
2. 匯入 OneComme 9.0
3. 完成設定
4. 開始使用

---

## 自訂修改

本專案提供完整原始碼。

若具備 HTML、CSS、JavaScript 基礎，可自行修改：

- 字體
- 顏色
- 動畫效果
- 版面配置

目前未提供圖形化設定介面，所有修改需直接編輯原始碼。
